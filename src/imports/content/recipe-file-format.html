recipe file format

The file format is a simple JSON format - an example of which is below: 


{static ../recipease/src/test/resources/breakfast/pancakes.json}

keys

For each of the keys on the root recipe object:

  • title This is the title of the recipe book - this defaults to I never knew what to call this...
  • information (optional) An array of JSON objects with key values
  • ingredients The ingredients that go into making the recipe.
  • directions How to make the recipe
  • notes (optional) Anything of interest about the recipe

ingredients

Ingredients can be further broken into sections, for example if you are making a cake and there is one set of ingredients for the cake and, and another set for the icing.

To break up the ingredients, at the start of the ingredient - place a section character ยง and this will highlight the text as separator.

directions

Directions can also be further broken into sections, for example if there are multiple things that need to be done in parallel, you can break up the directions into sections
